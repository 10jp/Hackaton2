Etiquetas de secciones

Body

Representa el contenido principal de un documento HTML.

Section

Define la seccion de un documento.

Article

Define contenido autonomo que podria existir independiente al resto

Header

Define la cabecera de una pagina.

Footer

Define el pie de pagina o seccion.

Main

Define el contenido principal o importante del documento. Solamente exite un elemento en el documento.

Etiquetas de Agrupacion

  • ul : Define una lista de articulos sin orden.
  • ol: Define una lista ordenada de articulos.
  • div: Representa un contenedor generico sin ningun significado especial
  • p: Define una parte que debe mostrarse como un parrafo.

    • Etiquetas de Contenido Inscrustado

    Img
  • Representa una imagen
    • Video
  • Representa un video y sus archivos asociados.
    • Map
  • Define un mapa de imagen
    • Math
  • Define una formula matematica

    • Etiquetas de Formularios

    input: Representa un campo de datos escrito que permite al usuario editar datos

    button: Representa un boton

    label: Representa el titulo de un control de formulario

    progress: representa el progreso de finalizacion de una tarea
